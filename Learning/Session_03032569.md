# Trading Roadmap (จาก Context ทั้งหมดใน Project Investing)

---

## Phase 1: Foundation – โครงสร้างตลาด & แนวคิดหลัก

### 1. Market Structure
- เข้าใจ HH / HL / LH / LL
- แยกแนวโน้ม (Uptrend / Downtrend / Sideway)
- ระบุ Internal Structure vs External Structure
- วาง Bias จาก HTF ก่อนเสมอ

### 2. BOS & CHOCH
- เข้าใจลำดับการเกิด:
  - CHOCH = สัญญาณเปลี่ยนแนวโน้มเบื้องต้น
  - BOS = การยืนยันแนวโน้มใหม่
- ฝึกแยก Minor vs Major BOS
- ใช้ BOS เป็น Confirmation ไม่ใช่จุด FOMO

---

## Phase 2: Multi-Timeframe Execution Model

### 1. Top-Down Analysis
ลำดับ Timeframe:
- H4 → หา Bias หลัก
- H1 → หาโครงสร้างย่อยตาม Bias
- M30 → หา Setup Zone
- M15 → Entry Trigger

### 2. Entry Logic
- เข้าเฉพาะเมื่อ:
  - ราคามาถึงโซนที่วางแผน
  - โครงสร้างย่อยสอดคล้องกับ Bias ใหญ่
  - มี CHOCH/BOS ยืนยันใน TF เข้า

- ห้ามเข้าเพราะ:
  - กลัวตกรถ
  - อยากแก้มือ
  - อารมณ์

---

## Phase 3: Risk Management System

### 1. Stop Loss Framework
- SL วางตาม Structure จริง ไม่ใช่ตามความสบายใจ
- ถ้า SL เกิน Risk Plan = ไม่เข้า
- ห้ามขยับ SL เพราะกลัว

### 2. Break Even Logic
- ขยับ BE เมื่อ:
  - โครงสร้างใหม่เกิดและยืนยัน
  - ราคาเคลื่อนผ่าน Internal Liquidity
- ห้ามขยับ BE เพียงเพื่อ “สบายใจ”

### 3. Daily Rules
- กำไรถึงเป้า = หยุดเทรด
- ขาดทุนถึง Risk Limit = หยุดเทรด
- ไม่ revenge trade

---

## Phase 4: Backtest → Forward Test → Live

### 1. Backtest
- ทดสอบระบบจนมี:
  - Winrate ชัดเจน
  - RR เฉลี่ย
  - Max Drawdown
- บันทึกสถิติทุกไม้

### 2. Forward Test
- เทรดบัญชีเล็ก / demo
- ฝึก Execution ตามแผนเป๊ะ
- ห้ามปรับแผนระหว่างทดสอบ

### 3. Live Trading Discipline
ปัญหาที่พบ:
- ระบบดี แต่เงินจริงเสีย
- ลังเลตอนเข้า
- อยากขยับ SL เป็น BE เร็วเกินไป
- หาที่พึ่งคนอื่นเมื่อไม่มั่นใจ

สิ่งที่ต้องพัฒนา:
- ความเชื่อในสถิติ
- ทำตามแผนแม้จะกลัว
- ไม่ seek validation จากคนอื่น

---

## Phase 5: Psychology & Behavioral Control

### 1. Emotional Awareness
- รู้ตัวเมื่อ:
  - กลัวเสีย
  - อยากเอาคืน
  - อยากเข้าเพราะเห็นกราฟสวย

### 2. Identity Shift
จาก:
- เทรดเพื่อชนะไม้

ไปเป็น:
- เทรดเพื่อ Execute ระบบ

### 3. No External Dependency
- หยุดพึ่ง signal / คนอื่น confirm
- เชื่อแผนที่ backtest มาแล้ว

---

## Phase 6: Journaling & Review System

### ทุกไม้ต้องบันทึก:
- TF Context
- เหตุผลเข้า
- โครงสร้างก่อนเข้า
- อารมณ์ก่อนเข้า
- ทำตามแผนครบไหม

### Review รายสัปดาห์
- เสียเพราะระบบ หรือวินัย?
- ขยับ SL ผิดกี่ครั้ง?
- เข้าไม้ที่ไม่อยู่ในแผนกี่ไม้?

# Core Rules (สรุปแก่นทั้งหมด)

1. Bias มาจาก HTF
2. Entry ต้องมี Structure Confirm
3. SL ตามโครงสร้างจริง
4. ไม่ขยับ SL เพราะอารมณ์
5. กำไรถึงเป้า = หยุด
6. ระบบชนะในระยะยาว ไม่ใช่ทุกไม้
7. Execute > Emotion
