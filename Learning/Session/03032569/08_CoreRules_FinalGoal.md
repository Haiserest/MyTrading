# Phase 8 – Core Rules & Final Goal

## Learning Objectives
- ทบทวนกฎหลัก 7 ข้อที่คุมทุกการตัดสินใจเทรด
- เชื่อมกฎเหล่านี้เข้ากับระบบที่สร้างใน Phase ก่อนหน้า
- กำหนดเป้าหมายสุดท้ายของระบบที่วัดผลได้ ทำซ้ำได้ และพึ่งตนเองได้

## Introduction
หลังผ่านทุก Phase ต้องสกัด “แก่น” เพื่อใช้เป็นเช็กลิสต์ก่อนเทรดทุกครั้ง Core Rules เหล่านี้จะช่วยป้องกันการลืมวินัยสำคัญ เช่น Bias, Confirmation, SL, Daily Rules และ Mindset เพื่อให้ระบบคงความแข็งแรงเมื่อเจอสภาวะตลาดจริง

## Core Concepts
1. **Bias มาจาก HTF** – เริ่มวิเคราะห์จาก Higher Timeframe เพื่อตั้งกรอบใหญ่ ไม่ให้สัญญาณ LTF หลอก[3]
2. **Entry ต้องมี Structure Confirm** – ใช้ CHOCH/BOS หรือสัญญาณที่ระบุไว้ เพื่อเลี่ยงการเข้าเพราะอารมณ์หรือ FOMO[3]
3. **SL ตามโครงสร้างจริง** – Stop Loss ต้องอิงจุดที่ทำให้ setup พัง ไม่ใช่ระดับที่รู้สึกสบายใจ[4]
4. **ไม่เลื่อน SL เพราะกลัว** – การขยับไป BE หรือดึง SL ขึ้นลงโดยไร้หลักฐาน ทำให้ระบบเสียรูปและถูกกวาดง่าย[5]
5. **Daily Rules: กำไรถึงเป้า/ขาดทุนถึง Limit = หยุด** – ป้องกันวันพัง ๆ ที่ทำลายผลงานทั้งเดือน[6]
6. **ระบบชนะในระยะยาว ไม่ใช่ทุกไม้** – เชื่อสถิติจาก Backtest/Forward Test และอย่าปล่อยให้การแพ้ครั้งเดียวสั่นวินัย[7][8]
7. **Execute > Emotion** – ใช้ Journal และกฎหยุดเพื่อปิดวงจร revenge trade เน้นทำตามแผนมากกว่าสนองอารมณ์[9][10][11]

## Practical Example
ก่อนเปิดออเดอร์ ให้ผ่านเช็กลิสต์ 7 ข้อด้านบน หากข้อใดตอบว่า “ไม่” ให้หยุดและทบทวน ไม่ว่าจะเป็นการตั้ง Bias, ตรวจ confirmation, หรือเช็ก Daily Loss Limit หากเข้าแล้ว ให้บันทึกใน Journal และหลังจบวันให้รีวิวว่าปฏิบัติตามกฎครบหรือไม่

## Common Mistakes
- มีแผนแต่ไม่ตรวจสอบก่อนกดคำสั่ง ทำให้เข้าไม้ที่ขัดกับกฎข้อใดข้อหนึ่ง
- คิดว่ากฎตั้งไว้ “เผื่อ ๆ” แล้วค่อยปรับหน้างาน ส่งผลให้ระบบไม่มีมาตรฐาน
- ไม่จดบันทึกหลังเทรด ทำให้ไม่รู้ว่าตัวเองละเมิดกฎข้อไหนบ่อยที่สุด

## Advanced Insight (Optional)
นำ Core Rules ไปเชื่อมกับ KPI ส่วนตัว เช่น % ไม้ที่ทำตามกฎครบ, จำนวนครั้งที่หยุดตาม Daily Limit, หรือจำนวนครั้งที่ Journal ชี้ว่าเกิด revenge trade แล้วปรับแผนเพื่อให้ตัวเลขเหล่านี้ดีขึ้นต่อเนื่อง

## Summary
Core Rules คือเกราะป้องกันระบบ เมื่อย้ำ 7 ข้อนี้ก่อนทุกการตัดสินใจ ระบบจะรักษามาตรฐานทั้งด้านโครงสร้าง ความเสี่ยง และจิตวิทยา เป้าหมายสูงสุดคือระบบที่วัดผลได้ ทำซ้ำได้ คุมความเสี่ยงได้ และตัดสินใจได้ด้วยตัวเอง โดยไม่ต้องพึ่งปัจจัยภายนอก
